# Web Scraper 3.0

A powerful web scraping tool that combines real-time search, AI-powered content parsing, and data storage capabilities.

## Overview

Web Scraper 3.0 is designed to perform real-time web scraping and data processing using AI models. It features a user-friendly interface built with Streamlit, allowing users to scrape websites, parse content using AI, and chat with the scraped data.

## Features

- **Real-time Web Scraping**: Scrape websites using Selenium with anti-scraping measures.
- **Google Search Integration**: Perform Google searches and scrape multiple results.
- **AI-powered Content Parsing**: Extract specific information from scraped content using LLMs.
- **Multi-model Support**: Process data with various AI models including Ollama, Groq, and OpenAI.
- **Data Storage**: Save raw and processed data in structured formats.
- **Interactive Chat**: Chat with the AI about the scraped data.
